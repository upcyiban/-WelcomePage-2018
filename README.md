```bash
ruby（gulp用ruby写的）

```
## 本地运行这个项目

> 首先安装 yo bower gulp
> (如果你很慢的话可以考虑使用换淘宝源或者使用 cnpm)

```bash
npm install gulp -g
npm i -g yo bower 

# 或者使用yarn(学长说的，我没用过)
yarn global add yo bower gulp
```

> 安装所需依赖

```bash
npm i
bower i
```

> 启动 gulp 服务预览

```bash
gulp serve
```

> 打包

```bash
gulp build
```
